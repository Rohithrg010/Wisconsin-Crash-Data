# Wisconsin Crash Data Visualization Project

This project analyzes Wisconsin traffic crash data to uncover patterns related to accident frequency, severity, location hotspots, weather conditions, and seasonal trends. The goal is to transform raw crash records into meaningful insights that can inform road safety awareness, infrastructure planning, and policy decisions.
